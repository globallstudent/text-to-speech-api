About This Project

Project Overview

This Text-to-Speech (TTS) web application converts written text into spoken audio using multiple high-quality speech synthesis engines, including Microsoft Edge TTS, Google TTS, and offline PyTTSx3.

Features

  • Convert text to natural-sounding speech
  • Choose from multiple TTS engines
  • Select from a wide variety of voices
  • Adjust speech speed
  • Download generated audio files
